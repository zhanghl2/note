**八、\*\***重置\***\*与变基**

**1.reset(重置)**

a.head

25:11

head: 当前的提交

head~: 上次的提交

head~2: 倒数第二次的提交

b. --soft

25:34

仅取消 commit 操作, 把修改文件暂存.

如果不加 --soft 则表示恢复到暂存前, 修改的内容是存在的.

c. --hard

26:01

取消暂存, 还取消修改内容, 彻底回到上次提交的状态.

不推荐使用, 可能丢失数据.
